Good / Bad Sitemap

Home

To Top

Home - Good

  • 01. Page properly titled
  • 02. Zooming and scaling content not prevented
  • 03. Skip-links to bypass repeated content
  • 04. Landmarks and sectioning elements used
  • 05. Primary Language of and within the Document is provided
  • 06. Main menu with NAV element, a list and proper attributes
  • 07. Search with NAV element, specific role attribute and a proper label
  • 08. Headings marked up as such
  • 09. Image with text alternative
  • 10. Image with 'null' text alternative
  • 11. Links and buttons highlighted when they receive focus
  • 12. Link text indicates the link purpose
  • 13. Links are Links and Buttons are Buttons
  • 14. Lists marked up as such and given an accessible name
  • 15. Sufficient color contrast

Home - Bad

  • 01. Page unclear titled
  • 02. Zoom and scaling prevented
  • 03. No mechanisms to bypass repeated content
  • 04. No Landmarks or sectioning elements used
  • 05. Omitted Language of and within the Document
  • 06. Main menu without name, role, states and values
  • 07. Omitted sectioning landmark, label for input and button not present
  • 08. Headings not marked up as such
  • 09. Informative image without any text alternative
  • 10. Decorative image without empty text alternative
  • 11. Links and buttons not properly highlighted when they receive focus
  • 12. Link text is not descriptive
  • 13. Links and Buttons are wrongly mixed for interactivity
  • 14. Lists not marked up as such
  • 15. Insufficient color contrast

Article

To Top

Article - Good

  • 01. Content provides a logical and hierarchical heading structure
  • 02. Correct use of lead paragraph
  • 03. Links are Links and Buttons are Buttons
  • 04. Max width of links, lists, input fields and paragraphs
  • 05. Accessibility and DoD are spelt out in full
  • 06. Headings describe topic or purpose
  • 07. Emphasized text marked up as such using structure to convey semantic meaning
  • 08. Links with distinctive appearance and communicating function
  • 09. Figure element with proper role and label
  • 10. Lists marked up as such and given an accessible name
  • 11. Link text visually distinct from its surrounding
  • 12. Link text descriptive on its own
  • 13. Only the first letter of words capitalized
  • 14. Centrally aligned text is avoided
  • 15. Full justification for text is avoided

Article - Bad

  • 01. Content does not provides a logical and hierarchical heading structure
  • 02. Incorrect use of a lead heading
  • 03. Links and Buttons are wrongly mixed for interactivity
  • 04. Max width of links, lists, input fields and paragraphs
  • 05. Acronym used to indicate accessibility is not explained
  • 06. Headings not describing topic or purpose
  • 07. Using styling elements or CSS to convey semantic meaning
  • 08. Links without communicating function and advanced warning when opening a new window
  • 09. Informative image without the Figure grouping element
  • 10. Lists not marked up as such
  • 11. Link text not visually distinct from its surrounding
  • 12. Link text not descriptive on its own
  • 13. Using all capital letters for words or sentences
  • 14. Centrally aligned text is applied
  • 15. Text is fully justified
